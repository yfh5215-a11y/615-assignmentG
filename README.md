# British Literary Prizes (1990–2022) — Shiny App

This repository contains a Shiny web application for exploring the **Selected British Literary Prizes dataset (1990–2022)** from **TidyTuesday (2025-10-28)**.

The app provides an interactive interface for examining trends related to:
- Prize genres  
- Representation of gender and ethnic groups  
- Prize outcomes (shortlisted vs. winners)  
- Differences across literary categories  



To run the app:

```r
library(shiny)
shiny::runApp()
